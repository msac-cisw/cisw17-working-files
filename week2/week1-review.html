Main elements of an HTML page

  1. <!DOCTYPE html>: Always needed to declare the document as an HTML5 page.
  2. <html>: The root of every HTML page, it must always be present.
  3. <head>: Technically, the <head> is not required, but it’s where you include metadata like the character set, which is important for correct rendering.
    • <title> inside the <head>: Required so that the browser shows a title in the tab.
  4. <body>: Required to hold the visible content of your page.

HTML Tags

HTML tags are keywords that are used to instruct web browsers on how to format and display a web page. They are made up of elements and attributes, and are enclosed in angle brackets, < and >.
